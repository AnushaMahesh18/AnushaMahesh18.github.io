About Me

Based in Rochester, NY, I’m an MS Computer Science student at Rochester Institute of Technology graduating May 2027, focused on building AI-powered, cloud-native software systems that work reliably in production. My recent projects include TrendPulse, a serverless NLP platform on AWS for real-time PR sentiment analysis and crisis detection, and FedPer, a personalized federated learning framework for handling non-IID data across distributed clients in PyTorch.

I care as much about the engineering behind ML systems as the models themselves—whether it’s event-driven architecture, infrastructure as code, observability, or bridging the gap from prototype to production-ready application. Through advanced coursework in Artificial Intelligence, Machine Learning, Computer Vision, Algorithms & Complexity, and Engineering Cloud Software Systems, I’m deepening my skills in distributed systems, AWS architectures, and integrating managed AI services like Amazon Comprehend into full software pipelines.

I want to join teams building intelligent, connected software—where AI is a tool, engineering quality drives impact, and ideas make it to real users.

What I’m building toward

  • AI-powered software: integrating ML models and managed services like PyTorch and Amazon Comprehend into production pipelines.
  • Cloud-native architecture: designing event-driven, serverless AWS systems with Lambda, DynamoDB, and API Gateway, provisioned with Terraform.
  • Distributed ML: federated learning, model aggregation, and scaling ML across distributed clients.
  • Production engineering: CI/CD with GitHub Actions, infrastructure as code, and robust monitoring with CloudWatch and SNS.
